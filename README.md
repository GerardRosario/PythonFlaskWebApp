# PythonFlaskWebApp

A lightweight web application used to create notes. 

## Technology / Tools

- Python
- Flask (Login, SQLAlchemy)
- VS Code Studio

## Features
- Sign Up
- Authentication
- Database initialization
- Add Notes / Delete Notes
